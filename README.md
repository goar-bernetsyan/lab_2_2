# API Object Detection
API приложения для детекции объектов на изображениях с использованием модели YOLOv5.

### Ссылка на модель
https://drive.google.com/file/d/1wEmok7Am8QvCDELcaVfCcCXxLftgPdOz/view?usp=drive_link

### Инструкция

1. Установить необходимые зависимости: pip install -r requirements.txt
2. Запустить сервер командой: uvicorn app:app --reload

Теперь API доступно по адресу `http://127.0.0.1:8000/detect`, куда можно отправлять POST-запросы с изображением для детекции

*ПРИМЕЧАНИЕ: при наличии интернета модель автоматически установится, в ином случает перед запуском приложения скачайте модель по ссылке и поместите её в корень проекта.*

### Как сделать запрос
1. Используя curl (в командной строке)
   
    curl -X POST "http://127.0.0.1:8000/detect" -F "file=@path"
   
    где вместо path вставьте полный путь к своему изображению (C:\...\image.jpg)
   
2. Используя Postman
   
    Откройте Postman.
   
    Создайте новый POST-запрос по адресу: http://127.0.0.1:8000/detect.
   
    Вкладка Body → выберите form-data.
   
    Введите ключ: file.
   
    В значение выберите файл изображения.
   
    Нажмите Send.
